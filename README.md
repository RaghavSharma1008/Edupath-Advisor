# EduPath Advisor

EduPath Advisor is a Streamlit-based web application designed to provide personalized career and education guidance to students after Class 10th and 12th.

## Problem Statement
Many students struggle to choose the right academic stream and career path due to lack of structured guidance and awareness about available government colleges.

## Features
- Login & Signup Authentication (CSV-based)
- Interest-Based Career Quiz
- Career Roadmap Generator
- Government College Finder
- Insights Dashboard
- Future Scope Planning

## Tech Stack
- Python
- Streamlit
- Pandas
- CSV-based Data Storage

## How to Run

1. Install dependencies:
   pip install -r requirements.txt

2. Run the application:
   streamlit run app.py
